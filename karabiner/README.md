# 65% keymap for Mac

## Function Keys
### F1 - F6
| Key | Function |
| ----- | ----- |
| F1 | Brightness Up |
| F2 | Brightness Down |
| F3 | Chrome (recent) |
| F4 | TickTick |
| F5 | Notion (recent) |
| F6 | reMarkable |

| Key | Function |
| ----- | ----- |
| F7 | Previous |
| F8 | Pause |
| F9 | Next |
| F10 | Volume Mute |
| F11 | Volume Up |
| F12 | Volume Down |
